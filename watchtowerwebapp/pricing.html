Frequently Asked Questions
What states do you cover? +
All 10 Day-1 states: California (BSIS), Florida (FDACS), Texas (DPS-PSB), Illinois (IDFPR), Virginia (DCJS), Nevada (PILB), Oregon (DPSST), Washington (DOL), Arizona (DPS), and North Carolina (PPSB). Data is pulled in real-time from official state regulatory databases. Additional states are on the roadmap.
What license types can you verify? +
We verify all individual guard credentials (Guard Cards, Class D/G licenses, PERC cards, Level II/III/IV commissions, Work Cards, etc.), firearm/armed endorsements, and company/agency licenses (PPO licenses, Class B, etc.). Each state's specific credentials are documented in our state coverage guide.
How fresh is the data? +
Individual lookups are real-time — we query the state's portal or API at the time of your search. Roster re-verification (for tracked guards) is refreshed every 24 hours. California uses an official API; other states use their public verification portals.
Can I cancel anytime? +
Yes. Cancel anytime from the billing portal — your access continues until the end of your current billing period. No cancellation fees. Annual plans are not refunded for unused months, but you can cancel to prevent renewal.
What counts as a "search"? +
Each individual license verification counts as one search. If you upload a roster of 50 guards, that counts as 50 searches. Results are cached for 30 minutes, so re-checking the same license within that window doesn't consume an additional search.
Do you offer refunds? +
We offer a 7-day money-back guarantee for new subscriptions. If you're not satisfied in your first 7 days, contact support for a full refund. After 7 days, we do not issue prorated refunds for the current billing period.
Is there an API for integration into our own system? +
Yes — the Enterprise plan includes REST API access with API key authentication. See our API docs for full reference. The API supports single-license verification, name search, and batch roster endpoints.
What payment methods do you accept? +
We accept all major credit/debit cards (Visa, Mastercard, Amex, Discover) via Stripe. Business and Enterprise customers can also pay by ACH bank transfer or request net-30 invoicing — contact sales.
